Personal Projects

What good is learning something if I can't apply it somewhere!

NBA Game Predictor
  • Scraped historical data of advanced statistics from all NBA teams as well as outcomes of their games in multiple seasons
  • Used data in conjunction with various classifying models to predict the outcomes of games during the 2019-2020 NBA season
  • Models produced an 82% accuracy for its output
  • This project was completed using python and libraries such as scikit-learn, numpy, and pandas





DFS/BFS Visualization



  • Programmed a simple visual demonstration of Depth First Search and Breath First Search algorithms on a random maze.
  • Implemented using python and the pygame module to create the GUI aspect





Spotify Rewind Web App


  • Use of the Flask library, HTML, and CSS to create the frontend website design
  • Spotipy API library used to authenticate user entry into the website and retrieve user data from their spotify account.
  • Data was used to display user's listening history over different periods of time, and a popularity score was given based on the popularity of their listening history.
  • Heroku used to push and host the website
  • Website: https://spotify-re.herokuapp.com/



NBA All-Star Predictor



  • Scraped historical data from individual players for their stats in their first 5 years in the league and also whether or not they made an All-Star Team in their career
  • Used this training data to predict whether current young players would become All-Star players in their future
  • Used python, pandas, and scikit-learn libraries to scrape data, fit data to models, explore correlations in data, and make predictions on the data
