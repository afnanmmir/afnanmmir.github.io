ECE Courses
  • EE302H - Intro to Electrical Engineering Honors
    • Fundamental techniques in circuit analysis such as mesh and node analysis and superposition
    • Concepts in Thevenin and Norton equivalent circuits and its applications such as max power transfer.
    • Workings of and functionality of different configurations of operational amplifiers
    • Topics specfic to the honors class include 2-port networks and bipolar junction transistors
  • EE306 - Intro to Computing
    • Provides a bottom-up approach or understanding the complexity of modern computers using the LC-3 assembly architecture
    • Introduction to low level bit gates such as decoders, full adders, and multiplexers
    • Fundamentals of finite state machines and their implementations in modern day systems
  • EE319K - Intro to Embedded Systems
    • Introduction to how computers and electronic devices are able to interact with their environment through input and output drivers
    • Use of ARM assembly, C, and C++ languages to program TM4C Microcontroller to perform various tasks including a finite state
      machine traffic light, and a small video game.
    • Introduction to communication interfaces such as SPI and UART as well as interrupts and their handling.
  • EE411 - Circuit Theory
    • Review of and expansion on operational amplifiers
    • Analysis of first order RC and RL circuits using inuductors, resistors, and capacitors
    • Analysis of second order RLC circuits and the various responses given different dampening conditions.
    • Sinusodial steady state analysis, three phase circuit analysis, and AC power analysis
    • Expansion on two-port networks and use of bode plots
  • EE312 - Software Design and Implementation I
    • Use of C and C++ programming languages to help learn various topics in software design
    • Review and expanding upon topics such as Object Oriented Programming, Recursion, and Big O Notation
    • Introduction to data structures such as linked lists, stacks, trees, and hashmaps
    • Use of heap memory allocation functions (malloc, realloc, free)
    • Implementation of algorithms such as sorting algorithms and binary search tree algorithms
